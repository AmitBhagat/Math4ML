Introduction to Linear Algebra for Applied Machine Learning with Python
Linear algebra is to machine learning as п¬Ӯour to bakery: every machine learning model is based in linear algebra, as every
cake is based in п¬Ӯour. It is not the only ingredient, of course. Machine learning models need vector calculus, probability, and
optimization, as cakes need sugar, eggs, and butter. Applied machine learning, like bakery, is essentially about combining these
mathematical ingredients in clever ways to create useful (tasty?) models.
This document contains introductory level linear algebra notes for applied machine learning. It is meant as a reference
rather than a comprehensive review. If you ever get confused by matrix multiplication, donвҖҷt remember what was the 
 norm, or
the conditions for linear independence, this can serve as a quick reference. It also a good introduction for people that donвҖҷt need
a deep understanding of linear algebra, but still want to learn about the fundamentals to read about machine learning or to use
pre-packaged machine learning solutions. Further, it is a good source for people that learned linear algebra a while ago and need
a refresher.
These notes are based in a series of (mostly) freely available textbooks, video lectures, and classes IвҖҷve read, watched and
taken in the past. If you want to obtain a deeper understanding or to п¬Ғnd exercises for each topic, you may want to consult those
